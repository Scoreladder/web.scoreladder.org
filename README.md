# Contributing
## For outsiders: 

Fork, and base your branch on `Scoreladder/www.scoreladder.org/develop`.

Then PR into develop.

To view live changes, run `python -m http.server 8080` and navigate to `localhost:8080` in your browser.


## For Scoreladder devs:

CREATE A NEW BRANCH BASED ON `develop`. **DO NOT FORK TO YOUR ACCOUNT**

Either run `python -m http.server 8080` as explained above;

OR publish your branch and navigate to `dev.scoreladder.org/branchname` to view live changes.

You must verify functionality based on `dev.scoreladder.org`, not your localhost.
